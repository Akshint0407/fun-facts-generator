# Personal Facts Display App

A simple, interactive single-page web application that displays personal facts one at a time. Built with vanilla JavaScript, HTML, and CSS, this app demonstrates basic DOM manipulation and event handling.

## Features
- Clean, minimalist interface
- Responsive design
- Circular navigation through facts
- Smooth transitions between displays

## Technical Stack
- [HTML5](https://www.w3schools.com/html/)
- [CSS3](https://css-tricks.com/)
- [Vanilla JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Project Structure
personal-facts-app/
├── index.html
├── styles.css
└── app.js
## Quick Start
1. Clone this repository
2. Open `index.html` in your web browser
3. Click the "Show Next Fact" button to cycle through facts
